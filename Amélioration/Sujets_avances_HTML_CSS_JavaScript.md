# 10 sujets avancés en HTML, CSS et JavaScript

# HTML -- 10 sujets avancés

## 1. Web Components

Créer des composants HTML réutilisables grâce aux Custom Elements,
Shadow DOM et Templates.

## 2. Shadow DOM

Isole le HTML et le CSS d'un composant afin d'éviter les conflits de
styles.

## 3. Custom Elements

Permet de créer ses propres balises HTML (`<my-card>`).

## 4. Templates (`<template>`)

Contenu HTML non rendu pouvant être cloné dynamiquement.

## 5. Slots

Permettent d'injecter du contenu dans un Web Component.

## 6. ARIA

Attributs améliorant l'accessibilité des applications web.

## 7. Microdata / Schema.org

Ajoute des données structurées pour améliorer le référencement.

## 8. Lazy Loading

Chargement différé des images et iframes.

## 9. Progressive Web Apps

HTML servant de base aux applications web installables.

## 10. API HTML5

Canvas, Drag & Drop, Geolocation, Web Storage, History API, etc.

------------------------------------------------------------------------

# CSS -- 10 sujets avancés

## 1. CSS Grid avancé

Création de mises en page complexes.

## 2. Flexbox avancé

Alignements sophistiqués et comportements responsives.

## 3. Variables CSS

Création de thèmes dynamiques.

## 4. Container Queries

Adapter le style selon la taille du conteneur.

## 5. Cascade Layers (@layer)

Contrôle précis de la priorité des styles.

## 6. Animations complexes

Keyframes, transitions, timing-functions.

## 7. CSS Houdini

Étendre les capacités natives du moteur CSS.

## 8. Fonctions modernes

clamp(), min(), max(), calc().

## 9. Subgrid

Grilles imbriquées parfaitement alignées.

## 10. Sélecteurs avancés

:has(), :is(), :where(), :not().

------------------------------------------------------------------------

# JavaScript -- 10 sujets avancés

## 1. Closures

Fonctions conservant leur contexte lexical.

## 2. Prototype et héritage

Fonctionnement interne des objets JavaScript.

## 3. Event Loop

Gestion asynchrone du moteur JavaScript.

## 4. Promises

Traitement propre des opérations asynchrones.

## 5. Async / Await

Syntaxe moderne basée sur les Promises.

## 6. Modules ES6

Organisation du code avec import/export.

## 7. Proxy

Intercepter le comportement des objets.

## 8. Generators

Fonctions pouvant être interrompues puis reprises.

## 9. Design Patterns

Module, Observer, Singleton, Factory, MVC...

## 10. Web APIs

Fetch, WebSocket, IndexedDB, Service Workers, Web Workers.

------------------------------------------------------------------------

# Conseils d'apprentissage

Ordre recommandé :

1.  HTML avancé
2.  CSS avancé
3.  JavaScript avancé
4.  Web Components
5.  Progressive Web Apps
6.  Frameworks (React, Vue, Svelte)

La maîtrise de ces notions permet de développer des interfaces modernes,
performantes et professionnelles.
