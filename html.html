Les bases de HTML

HTML (HyperText Markup Language) décrit la structure d’une page web avec des balises.

1. Qu’est-ce que HTML ?

HTML n’est pas un langage de programmation : c’est un langage de balisage. Chaque élément a une balise ouvrante et souvent une balise fermante. 

<p>Bonjour le monde</p>

2. Structure d’une page

Toute page HTML commence comme ceci :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Ma page</title>
</head>
<body>
    <h1>Titre principal</h1>
    <p>Mon premier paragraphe.</p>
</body>
</html>
  • <head> : infos pour le navigateur (titre, CSS…)
  • <body> : contenu visible de la page

3. Titres et paragraphes

Il existe 6 niveaux de titres, de h1 (le plus important) à h6.

<h1>Titre principal</h1>
<h2>Sous-titre</h2>
<p>Un paragraphe de texte.</p>

Résultat

Titre principal

Sous-titre

Un paragraphe de texte.

4. Liens et images

<a href="https://exemple.com">Visiter le site</a>

<img src="photo.jpg" alt="Description de l'image">
  • href : destination du lien
  • src : chemin de l’image
  • alt : texte alternatif (accessibilité)

5. Listes

<!-- Liste à puces -->
<ul>
    <li>HTML</li>
    <li>CSS</li>
</ul>

<!-- Liste numérotée -->
<ol>
    <li>Ouvrir un éditeur</li>
    <li>Écrire du HTML</li>
</ol>

6. Formulaire simple

<form action="#" method="post">
    <label for="nom">Nom</label>
    <input type="text" id="nom" name="nom">

    <label for="email">Email</label>
    <input type="email" id="email" name="email">

    <button type="submit">Envoyer</button>
</form>
