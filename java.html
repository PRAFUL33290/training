Les bases de Java

Java est un langage de programmation orienté objet. Il sert pour les apps, le backend, Android et bien plus.

1. Qu’est-ce que Java ?

Contrairement à HTML et CSS, Java est un vrai langage de programmation : tu écris des instructions exécutées par la machine virtuelle Java (JVM).

  • Fichier source : MonProgramme.java
  • Compilation : javac MonProgramme.java
  • Exécution : java MonProgramme

2. Premier programme

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Bonjour Java !");
    }
}
  • Le nom de la classe doit correspondre au nom du fichier
  • main est le point d’entrée du programme
  • System.out.println affiche du texte

4. Conditions (if / else)

int note = 15;

if (note >= 10) {
    System.out.println("Réussi");
} else {
    System.out.println("À retravailler");
}

5. Boucles

// Boucle for : répéter un nombre de fois
for (int i = 1; i <= 5; i++) {
    System.out.println("Tour n°" + i);
}

// Boucle while : tant que la condition est vraie
int n = 0;
while (n < 3) {
    System.out.println(n);
    n++;
}

6. Tableaux

String[] langages = {"HTML", "CSS", "Java"};

System.out.println(langages[0]); // HTML
System.out.println(langages.length); // 3

for (String lang : langages) {
    System.out.println(lang);
}

7. Méthodes (fonctions)

public class Calcul {
    public static int addition(int a, int b) {
        return a + b;
    }

    public static void main(String[] args) {
        int resultat = addition(5, 3);
        System.out.println(resultat); // 8
    }
}

8. Classe et objet (intro)

public class Chien {
    String nom;

    void aboyer() {
        System.out.println(nom + " dit wouf !");
    }

    public static void main(String[] args) {
        Chien monChien = new Chien();
        monChien.nom = "Rex";
        monChien.aboyer(); // Rex dit wouf !
    }
}

Une classe est un modèle. Un objet est une instance de cette classe.
